# QuizCaju

Quiz interativo feito em HTML, CSS e JavaScript puro — projeto do CajuHub / Jovem Tech.

## Como rodar

Abra o arquivo `quiz-caju/index.html` no navegador.

## Estrutura

- `quiz-caju/index.html` — estrutura das telas (home, questão, feedback, resultado)
- `quiz-caju/S1_style.css` — estilos com identidade visual do caju (laranja/amarelo)
- `quiz-caju/S1_app.skeleton.js` — lógica do quiz
- `quiz-caju/S1_questions.js` — banco de perguntas

## Funcionalidades

- 4 telas: home, questão, feedback e resultado final
- Timer de 20s por questão
- Barra de progresso
- Placar e aproveitamento ao final
